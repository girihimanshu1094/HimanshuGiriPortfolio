Personal Portfolio Website

A simple and responsive personal portfolio website built using HTML, CSS, and JavaScript. This project showcases my skills, projects, education, and contact information in a clean and user-friendly interface.

Features
1. Responsive design for desktop, tablet, and mobile devices
2. Simple navigation bar with smooth scrolling
3. Hero section with introduction and resume download button
4. About Me section
5. Skills section
6. Projects showcase
7. Education section
8. Back-to-top button\
9. Basic JavaScript interactions and form validation

Technologies Used
1. HTML5
2. CSS3
3. JavaScript (Vanilla JS)
